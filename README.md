# 📰 NewsHub - News Aggregator

A modern news aggregation platform built with React and News API.

## ✨ Features

- 📰 Real-time news from multiple sources
- 🔍 Search functionality
- 📱 Responsive design
- 🌙 Dark mode
- 🔖 Bookmark articles
- 🌍 Multiple countries support
- 🎨 Modern glassmorphism UI

## 🚀 Tech Stack

- **Frontend:** React + Vite
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **API:** News API
- **Icons:** React Icons
